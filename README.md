# pink-patrick
Big data fun with presto!!

## Installation

for creating the project:
```bash
docker-compose up --build -d
```
to stop it
```bash
docker-compose down -v
```
## Usage

#### presto panel:
```bash
http://0.0.0.0:8080
```
#### tweeter streams:
```bash
make run-twitter
```

#### presto to mongodb:
```bash
make run-presto-mongo
```
#### presto to hive:
```bash
make run-presto-hive
```
